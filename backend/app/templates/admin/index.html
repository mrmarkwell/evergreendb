EvergreenDB Records View

Use this page to add records to EvergreenDB.

Instructions For Use

Select the tab for the record type you'd like to add, update, or delete

All records of this type will be displayed in a list

  • To create a new record of this type, select "Create" and fill in the fields. Don't forget to save when you're done!
  • To edit an existing record, select the pencil icon to the left of the record you'd like to edit.
  • To view details of a certain record, select the eye icon to the left of that record.
  • To export records to a CSV file, select "Export"
  • To delete a record, click the trash can icon to the left of the record.
  • To delete many records, check the box next to the records and select "With selecte -> delete"
